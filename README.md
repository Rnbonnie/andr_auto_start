# Autostart Video App 

Это приложение автоматически запускает видео при загрузке системы.

## Требования
- Видео файл должен находиться по пути: `/autostart/auto.mp4`
- Приложение должно иметь разрешение на чтение медиа-файлов.
- Для корректного автозапуска на некоторых прошивках может потребоваться разрешение "Отображение поверх других окон" (Display over other apps) и отключение оптимизации батареи.

## Как это работает
1. `BootReceiver` ловит событие `BOOT_COMPLETED`.
2. Запускается `MainActivity`.
3. `MainActivity` проверяет наличие файла в `/autostart/auto.mp4`.
4. Если файл найден, он запускается в циклическом режиме (looping).

## Пути к файлу
Приложение ищет файл в двух местах:
1. Прямо в корне: `/autostart/auto.mp4` (если есть права root или специфическая разметка).
2. Во внутренней памяти: `/sdcard/autostart/auto.mp4` (стандартный путь для большинства устройств).

## Разрешения
- `RECEIVE_BOOT_COMPLETED`
- `READ_MEDIA_VIDEO` (Android 13+)
- `SYSTEM_ALERT_WINDOW` (для обхода ограничений на запуск из фона)
